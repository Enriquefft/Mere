# Mere Roadmap

This roadmap outlines planned features and enhancements for Mere.

## v0.2 (Current)

### Installation & Distribution
- [x] Basic CLI installation via curl
- [x] Go install support
- [x] Nix development environment
- [ ] Homebrew tap for macOS/Linux
- [x] Installation script

### Core Features
- [x] `mere init` command
- [x] `mere version` command
- [x] `/module` command (create/work)
- [x] `/status` command
- [x] Language-agnostic scaffolding
- [x] Embedded templates

### Documentation
- [x] README with badges
- [x] Contributing guidelines
- [x] License
- [x] Code of Conduct

## v0.3 (Planned)

### CLI Enhancements
- [ ] Shell completion (bash, zsh, fish)
- [ ] `mere upgrade` command
- [ ] `mere doctor` command (diagnose issues)
- [ ] Color output support
- [ ] Verbose mode for debugging

### Template Improvements
- [ ] Additional command templates
- [ ] Custom template support (`mere init --template`)
- [ ] Interface validation (lint INTERFACE.md)

### Quality of Life
- [ ] GitHub Actions for all platforms (Linux, macOS, Windows)
- [ ] Automatic binary uploads to releases
- [ ] Version detection and update notifications

## v0.4 (Planned)

### Advanced Features
- [ ] Module dependency graph visualization
- [ ] `mere graph` command
- [ ] Interface compatibility checker
- [ ] Module refactoring suggestions
- [ ] Test coverage reporting

### Integration
- [ ] VS Code extension (basic integration)
- [ ] Neovim plugin
- [ ] JetBrains plugin API documentation

## v0.5 (Planned)

### Community Features
- [ ] Template marketplace (community-contributed)
- [ ] Module library (shareable modules)
- [ ] Success story showcase
- [ ] Newsletter signup

### Analytics & Telemetry
- [ ] Anonymous usage analytics
- [ ] Error reporting (opt-in)
- [ ] Success metrics tracking

## v1.0 (Future)

### Visual Interface
- [ ] Web UI for project management
- [ ] Interactive module creation
- [ ] Visual dependency graph
- [ ] Real-time collaboration

### Enterprise Features
- [ ] Team templates
- [ ] Custom branding
- [ ] On-premises installation
- [ ] Enterprise support

## How This Roadmap Works

1. **Planned:** Features under consideration for implementation
2. **In Progress:** Currently being developed
3. **Shipped:** Released and available

Want to influence what gets built next?

- **Vote for features:** Add a 👍 reaction to roadmap items you want
